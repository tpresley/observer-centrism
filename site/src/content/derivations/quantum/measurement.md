## Statement

Measurement is a Type III interaction that generates a relational invariant between observer and system. What is called "collapse" is the transition from a state with no shared relational invariant to a state with a definite relational invariant. There is no physical collapse -- no sudden, nonlocal, or non-unitary process. There is only the generation of new relational structure, governed by the same coherence-conserving dynamics as all other interactions.

## Derivation

### Step 1: The Pre-Measurement State

Let observer $\mathcal{O}$ and system $S$ be two observers in the interaction graph $\mathcal{G}$ that have not yet undergone a Type III interaction with respect to the observable in question. The pre-measurement condition is:

**No relational invariant for the observable exists between $\mathcal{O}$ and $S$.**

This does not mean $\mathcal{O}$ and $S$ have never interacted -- they may share relational invariants from prior interactions involving other observables. It means specifically that no relational invariant $I_{\mathcal{O}S}$ corresponding to the observable being measured has been generated.

In this pre-measurement state, the system $S$ (relative to $\mathcal{O}$) is described by a superposition:

$$|\psi_S\rangle_\mathcal{O} = \sum_k \psi_k |k\rangle$$

where $\{|k\rangle\}$ is the eigenbasis of the relational invariant that **will be** generated by the upcoming interaction (determined by the [interaction configuration](/derivations/quantum/preferred-basis)), and the subscript $\mathcal{O}$ emphasizes that this description is relative to observer $\mathcal{O}$.

The amplitudes $\psi_k$ encode the coherence distribution across the possible outcomes, as established in the [Born rule derivation](/derivations/quantum/born-rule).

### Step 2: The Measurement Interaction

The measurement is a [Type III interaction](/derivations/interactions/three-types) between $\mathcal{O}$ and $S$. By definition:

1. Both $\mathcal{O}$'s invariant $I_\mathcal{O}$ and $S$'s invariant $I_S$ survive
2. A new [relational invariant](/derivations/interactions/relational-invariants) $I_{\mathcal{O}S}$ is generated on the joint state space

The interaction transforms the joint state as follows. Before:

$$|\Phi_{\text{before}}\rangle = |\text{ready}\rangle_\mathcal{O} \otimes \sum_k \psi_k |k\rangle_S$$

where $|\text{ready}\rangle_\mathcal{O}$ is the observer's state prior to the interaction. After the Type III interaction generates $I_{\mathcal{O}S}$:

$$|\Phi_{\text{after}}\rangle = \sum_k \psi_k |d_k\rangle_\mathcal{O} \otimes |k\rangle_S$$

where $|d_k\rangle_\mathcal{O}$ is the observer state correlated with system outcome $k$. The relational invariant takes the definite value $\lambda_k$ on each branch $|d_k\rangle_\mathcal{O} \otimes |k\rangle_S$.

This is the standard von Neumann measurement interaction, but here it is **derived** as a Type III process rather than postulated.

### Step 3: What "Collapse" Actually Is

From $\mathcal{O}$'s perspective, the measurement produces a definite outcome -- say, outcome $k_0$. The post-measurement description of $S$ relative to $\mathcal{O}$ is:

$$|\psi_S\rangle_\mathcal{O}^{\text{after}} = |k_0\rangle$$

The transition from $\sum_k \psi_k |k\rangle$ to $|k_0\rangle$ is what is traditionally called "collapse." In the framework, this transition has a precise structural interpretation:

**Before measurement:** No relational invariant $I_{\mathcal{O}S}$ exists for this observable. The system's state relative to $\mathcal{O}$ is a superposition -- $\mathcal{O}$ has no coherence-stable basis element to correlate with.

**After measurement:** The relational invariant $I_{\mathcal{O}S}$ exists and takes the definite value $\lambda_{k_0}$. The system's state relative to $\mathcal{O}$ is the eigenstate $|k_0\rangle$ corresponding to this value.

"Collapse" is the generation of the relational invariant $I_{\mathcal{O}S}$. It is:

- **Local**: The Type III interaction occurs at a definite location in the interaction graph $\mathcal{G}$
- **Unitary**: The joint state $|\Phi_{\text{after}}\rangle$ is related to $|\Phi_{\text{before}}\rangle$ by a unitary transformation on the joint state space (coherence is conserved)
- **Irreversible**: Once generated, $I_{\mathcal{O}S}$ is conserved ([Axiom 1](/derivations/axioms/coherence-conservation)). The relational invariant cannot be un-generated. This irreversibility is the same structural arrow that gives rise to [time](/derivations/thermodynamics/time).

There is no non-unitary "projection postulate." The apparent non-unitarity arises from describing the system's state **relative to one observer** rather than describing the full joint state.

### Step 4: Relativity of the Measurement Outcome

The relational invariant $I_{\mathcal{O}S}$ is a property of the $\mathcal{O}$-$S$ pair, not of $S$ alone. A third observer $\mathcal{P}$ who has not interacted with $S$ (and therefore has no relational invariant $I_{\mathcal{P}S}$ for this observable) still describes $S$ in a superposition:

$$|\psi_S\rangle_\mathcal{P} = \sum_k \psi_k' |k'\rangle$$

This is not an approximation or a statement about $\mathcal{P}$'s ignorance. It is the correct description of $S$ relative to $\mathcal{P}$, because the relational invariant $I_{\mathcal{P}S}$ does not yet exist.

The two descriptions are **simultaneously correct**:

| Observer | Relational invariant with $S$ | Description of $S$ |
|---|---|---|
| $\mathcal{O}$ (has measured) | $I_{\mathcal{O}S}$ exists, value $\lambda_{k_0}$ | Definite state $\|k_0\rangle$ |
| $\mathcal{P}$ (has not measured) | $I_{\mathcal{P}S}$ does not exist | Superposition $\sum_k \psi_k' \|k'\rangle$ |

There is no contradiction because "the state of $S$" is not an observer-independent quantity. It is always relative to a specific observer's relational invariants.

### Step 5: Wigner's Friend

The Wigner's friend scenario illustrates this relational structure precisely.

**Setup:** Observer $\mathcal{F}$ (the friend) measures system $S$ inside a sealed laboratory. Observer $\mathcal{W}$ (Wigner) remains outside and does not interact with $S$ or $\mathcal{F}$ regarding the measurement.

**After $\mathcal{F}$'s measurement:**

- $\mathcal{F}$ has a relational invariant $I_{\mathcal{F}S}$ with definite value $\lambda_{k_0}$. Relative to $\mathcal{F}$: $S$ is in state $|k_0\rangle$.
- $\mathcal{W}$ has no relational invariant with $S$ (for this observable) and no relational invariant with $\mathcal{F}$ that encodes the outcome. Relative to $\mathcal{W}$: the joint $\mathcal{F}$-$S$ system is in the entangled state $\sum_k \psi_k |d_k\rangle_\mathcal{F} |k\rangle_S$.

**Both descriptions are correct** relative to their respective observers. There is no paradox because:

1. $\mathcal{F}$'s definite outcome is encoded in $I_{\mathcal{F}S}$ -- a relational invariant that $\mathcal{W}$ does not share
2. $\mathcal{W}$'s superposition description is correct because $\mathcal{W}$ lacks the relational invariant that would select a definite outcome
3. If $\mathcal{W}$ subsequently performs a Type III interaction with $\mathcal{F}$ (asks what the result was), a new relational invariant $I_{\mathcal{W}\mathcal{F}}$ is generated, and $\mathcal{W}$ obtains a definite outcome -- consistent with $\mathcal{F}$'s prior result, because the relational invariant $I_{\mathcal{F}S}$ constrains the possible values of $I_{\mathcal{W}\mathcal{F}}$

This is the relational interpretation of quantum mechanics, but **derived from the axioms** rather than postulated as an interpretive stance.

### Step 6: Connection to Entropy

The measurement process has a direct connection to [entropy](/derivations/thermodynamics/entropy).

**Before measurement:** The system $S$ is in a superposition relative to $\mathcal{O}$. The coherence of $S$ that lies outside $\mathcal{O}$'s coherence domain contributes to $\mathcal{O}$'s entropy:

$$S_\mathcal{O}(S) = \mathcal{C}(S) - \mathcal{C}_\mathcal{O}(S) > 0$$

The superposition represents coherence that $\mathcal{O}$ cannot access -- it is "inaccessible coherence" in the sense of the entropy derivation.

**After measurement:** The relational invariant $I_{\mathcal{O}S}$ brings part of $S$'s coherence into $\mathcal{O}$'s coherence domain. The accessible coherence $\mathcal{C}_\mathcal{O}(S)$ increases by the coherence content of the relational invariant:

$$\mathcal{C}_\mathcal{O}^{\text{after}}(S) = \mathcal{C}_\mathcal{O}^{\text{before}}(S) + \mathcal{C}(I_{\mathcal{O}S})$$

Consequently, the entropy of $S$ relative to $\mathcal{O}$ decreases:

$$S_\mathcal{O}^{\text{after}}(S) = S_\mathcal{O}^{\text{before}}(S) - \mathcal{C}(I_{\mathcal{O}S})$$

This decrease is **information gain**: $\mathcal{O}$ has learned the value of the relational invariant, converting inaccessible coherence into accessible coherence. The decrease does not violate the second law, because the second law applies to the total entropy of a bounded observer (including the measurement apparatus, environment, etc.), not to the entropy of a single subsystem.

### Step 7: No Interpretive Baggage

The framework resolves the measurement problem without introducing any elements beyond the three axioms:

| Interpretation | Extra element | Framework |
|---|---|---|
| Copenhagen | Collapse postulate (non-unitary, instantaneous) | No collapse -- relational invariant generation is unitary and local |
| Many Worlds | Ontological branching of the universe | No branching -- one interaction graph, multiple relational perspectives |
| Bohmian mechanics | Hidden particle positions + guiding equation | No hidden variables -- relational invariants are the complete description |
| QBism | Subjective Bayesian probabilities | Probabilities are objective (Born rule from coherence conservation) but observer-indexed |
| Relational QM (Rovelli) | Postulated relationality | Relationality is derived from the axioms, not postulated |

The framework is closest in spirit to Rovelli's relational quantum mechanics, but differs in a crucial respect: RQM postulates that quantum states are relational without deriving why. The observer-centric framework derives the relational structure from [Coherence Conservation](/derivations/axioms/coherence-conservation), the [Observer Definition](/derivations/axioms/observer-definition), and [Loop Closure](/derivations/axioms/loop-closure). The relational invariants that encode measurement outcomes are constructed objects within the theory, not interpretive choices.

## Rigor Assessment

**What is established:**
- The Type III interaction structure (both invariants survive, new relational invariant generated) is well-defined from the [interaction classification](/derivations/interactions/three-types)
- The correlation structure $\sum_k \psi_k |d_k\rangle_\mathcal{O} |k\rangle_S$ follows from unitary evolution on the joint space -- this is the standard von Neumann measurement model
- The consistency of Wigner's friend follows from the relational invariant structure: different observers have different relational invariants, hence different (but compatible) descriptions
- The entropy connection (measurement as coherence domain expansion) is consistent with the [entropy derivation](/derivations/thermodynamics/entropy)

**What needs work:**
- The derivation assumes that the Type III interaction between $\mathcal{O}$ and $S$ produces the von Neumann correlation structure. This should be derived from the general properties of Type III interactions, not assumed
- The mechanism by which $\mathcal{O}$ "selects" a single outcome $k_0$ from the entangled state is not fully specified. The Born rule gives the probability of each outcome, but the framework should clarify what structural feature determines the actual outcome in a single trial
- The entropy decrease $\Delta S_\mathcal{O} = -\mathcal{C}(I_{\mathcal{O}S})$ needs to be reconciled with the second law. The full accounting (including the entropy cost of operating the measurement apparatus) should show that total entropy is non-decreasing

## Open Gaps

1. **The single-outcome problem**: The derivation explains why measurement outcomes are relative to observers and why probabilities follow the Born rule, but does not fully explain what determines the specific outcome $k_0$ in a single trial. This is the residual core of the measurement problem. The framework's position -- that the outcome is determined by the full coherence structure of the interaction, which is not accessible to any bounded observer -- needs further development.
2. **Objectivity of outcomes**: When multiple observers independently measure the same system in the same basis, they obtain consistent results (quantum Darwinism). The framework should derive this from the constraint that relational invariants $I_{\mathcal{O}_1 S}$ and $I_{\mathcal{O}_2 S}$ must be mutually consistent when $\mathcal{O}_1$ and $\mathcal{O}_2$ share relational invariants with each other.
3. **Delayed-choice and quantum eraser**: In delayed-choice experiments, the "measurement basis" appears to be selected after the system has already been detected. The framework should show that this is naturally accommodated: the relational invariant is generated at the time of the Type III interaction, regardless of when the observer "decides" which interaction to perform. There is no retrocausality because the partial order in the [interaction graph](/derivations/thermodynamics/time) is respected.
4. **Continuous measurement and weak measurement**: Not all measurements are sharp Type III interactions. Weak measurements, continuous monitoring, and partial measurements should correspond to Type III interactions that generate relational invariants with small coherence content $\mathcal{C}(I_{\mathcal{O}S}) \ll \mathcal{C}(S)$. This limit needs formalization.
5. **Extended Wigner's friend scenarios**: Recent no-go theorems (Frauchiger-Renner, Brukner) place constraints on the consistency of measurement outcomes across nested observers. The framework should be tested against these scenarios to verify that the relational invariant structure avoids the contradictions.
